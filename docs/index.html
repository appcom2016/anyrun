核心能力

从安全执行到自我进化，anyrun 为 AI Agent 提供完整的基础设施

🔒

零配置沙箱

Docker 隔离执行，一行代码 Sandbox.run() 即可运行任意 Python。自动管理容器生命周期，无需手动 docker run。

Sandbox
📊

全量执行轨迹

每次执行自动采集结果、时长、错误、Traceback。SQLite 索引 + JSON 持久化，支持查询、统计、导出。

ExecutionTrace
🔍

模式发现引擎

自动识别错误聚类、成功路径和异常模式。基于执行历史分析问题根因，为后续优化提供数据驱动依据。

PatternAnalyzer
🧠

LLM 经验提取

从已发现的模式中自动提取最佳实践，生成结构化的 SKILL.md 技能文档。包含触发条件、执行步骤、常见陷阱。

ExperienceExtractor
🔄

自进化闭环

技能自动经历 beta → prod → decayed → retired 生命周期。退化技能自动检测并触发 LLM 修复，形成持续改进闭环。

Evolution
🔌

MCP 协议集成

原生支持 MCP（Model Context Protocol）服务器。可直接接入 Hermes、Claude Desktop 等 AI Agent 平台。

MCP

📦 安装

Shell
pip install anyrun-agent

⚡ 基础使用

Python
from anyrun import Sandbox

# 零配置，一行代码在 Docker 中安全执行
with Sandbox() as s:
    result = s.run("print(1 + 1)")
    print(result.data)  # "2\n"

🧪 多会话隔离

Python
sandbox = Sandbox()

# 两个隔离的会话
sandbox.run("x = 42", session_id="session-a")
sandbox.run("y = 100", session_id="session-b")

# 会话内文件持久化
sandbox.run('open("/app/data.txt","w").write("hello")')
sandbox.cleanup_session("session-a")

🔧 工具注册与执行

Python
from anyrun import ToolRegistry, Tool

registry = ToolRegistry()
registry.add_tool(Tool(
    name="add_numbers",
    description="对两个数字求和",
    parameters={"a": {"type": "integer"}, "b": {"type": "integer"}},
    code="def execute_tool(a: int, b: int):\n    return a + b",
))

# 转换为 OpenAI Function Calling 格式
tools = [t.to_openai_tool() for t in registry.get_tools_info()]

数据架构

本地优先存储，无需外部数据库

~/.anyrun/
├── traces/
│ ├── data/ # 执行轨迹 JSON
│ ├── index.db # SQLite 索引
│ └── patterns/ # 发现的模式
├── skills/ # 自动提取的技能
│ └── <name>/SKILL.md
└── evolution/
└── evolution.db # 技能生命周期数据
